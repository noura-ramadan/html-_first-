      5. The PC computer architecture performance test utilized is comprised of 22 individual benchmark tests that are available in six test suites.
      The six different test suites test for the following:

      1. Integer and floating-point mathematical operations
      2. Tests of standard two-dimensional graphical functions
      3. Reading, writing, and seeking within disk files
      4. Memory allocation and access
      5. Tests of the MMX (multimedia extensions) in newer CPUs

      The test results reported are shown as relative values. The larger the number the faster the computer. For example, a computer with a result of 40 can process roughly twice as much data as a computer with a result of 20.
      The Passmark rating is a weighted average of all the other test results and gives a single overall indication of the computer's performance.
      The bigger the number the faster the computer.
